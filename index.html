Что такое блокчейн и как он работает

Технология блокчейн у многих вызывает ассоциацию с биткойном, криптовалютой, майнингом. Блокчейн - это технология которая позволяет хранить любые данные распределенно. Можно сравнить блокчейн с базой данных, отличие лишь в том, что база данных расположенна на одном сервере, тогда, как блокчейн может хранить ваши данные на многих серверах одновременно, что способствует надежности данных от потери и “чистоте информиции” .

Что нужно знать при подготовке к WSR

В 20 19 году при проведении WSR Blockchain нами был использован следующий стек: Geth, web3.js, Solidity и средства web разработки (html, css, js).

Web3.js - Ethereum JavaScript API

Web3.js - это API для взаимодействия с блокчейном и смарт-контрактом. Подключается как обычный скрипт к html странице.
